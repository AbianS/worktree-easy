## 0.0.1

### Features

- Initial release
- TreeView in Activity Bar for managing git worktrees
- Create worktrees from existing or new branches
- Open worktrees in new VS Code window
- Archive (delete) worktrees with confirmation
- Status bar showing current worktree
- Configurable worktrees folder location
- Post-create tasks system for automation
- Keyboard shortcuts (Cmd+Shift+W C/O/S)
